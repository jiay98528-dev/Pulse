## 二、技术栈

| 层 | 技术 | 选型原因 |
|---|---|---|
| UI 壳 | **Tauri v2** (Rust + WebView2) | Surface Go 4GB RAM，比 Electron 省 100MB+ |
| 前端 | **原生 HTML/CSS/JS** + **Chart.js 4.x** (CDN) | 无框架依赖，轻量，实时图表质量高 |
| 后端 | **Python 3.12+ FastAPI** + WebSocket | psutil/WMI/Deepseek API 只有 Python 版 |
| 数据采集 | **psutil** (系统) + **WMI** (Windows) + **aiohttp** (API) | 已在 collectors/ 封装 |
| 数据库 | **SQLite** (aiosqlite) | 零配置，深度求索历史持久化 |
| 通信 | **WebSocket** (每秒系统 / 每30秒 API) | 实时推送 |
| 图表 | **Chart.js 4.x** | CDN 加载，硬边折线/柱状/环形图 |

## 七、设计约束 (关键!)

> 来源: `docs/错题本.md` 和 `DESIGN.md`

1. **不简陋** — 每个几何块是精心构图, 不是省事产物
2. **不浮夸** — 无渐变/无发光/无毛玻璃/无圆滑边框
3. **不赛博** — 1920s 工厂美学, 不是 2049 夜店美学
4. 红/黑/白三色系统占 90%+, 红色永不作大面积背景
5. 直角优先 (`border-radius: 0`)
6. 硬阴影 (`box-shadow` 仅偏移无模糊)
7. 图表最多三色 (红/黑/灰 + 必要时黄)
8. 标题 ALL CAPS, 等宽字体用于数值
9. 最小字号 14px, KPI 数值 56px, 触屏热区 >= 44px
